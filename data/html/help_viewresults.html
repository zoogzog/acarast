Viewing results


Viewing images
The are several ways to check the results of cell detection and feature extraction. Detected samples could be viewed by selecting them from the sample box in the Project manager. The detected cells could be viewed by choosing the Show detected cells or Show colored cells options in the View menu. The first method will draw boundary boxes around detected cells and the second one will colored cells on a black background. The image with boundary boxes could also be viewed by clicking the View cells button ().

Extracted features Features, extracted from the detected cells could be viewed in the Output viewer. All the features are divided into three categories: length frequency histogram, absolute features and relative features. The length frequency histogram represents the frequency of cells of each particular length. Absolute features are features extracted from the sample and relative features are sample features in relation to control sample features.

Absolute features
  • Cell count total amount of detected cells
  • Cell length features are average, minimum and maximum cell lengths
  • Interval is a confidence interval, where X amount of bacteria with the length in this interval are located. X is specified by the parameter Confidence interval in the Settings menu.
  • Cell area features are an average, minimum and maximum cell areas
  • Average width is an average cell width
  • Length deviation is the standard deviation of the cell lengths
  • Total pixel density is a relation between amount of pixels which are cells and total amount of pixels of the selected sample
  • Shape descriptor represents how close the cell shape to a line. The close it is the closer the value to 1.

Relative features
  • Total cell count is a cell count of the sample divided on cell count of the controls sample
  • Equal to control is amount of cells from the sample which length is in the confidence interval of the control sample
  • Average length is test sample average length divided on control sample average length
  • Length deviation is length deviation in relation to the control sample
  • Average area is the average area of the test sample divided on average area of the control sample
  • Total pixel density is the pixel density of the test sample divided on the pixel density of the control sample
  • Interval length is a relation between the test sample confidence length interval and the control sample interval
  • Shape descriptor is a test sample shape descriptor divded on the control sample descriptor
  • Level of difference represents how different is the test sample from the control sample. The closer the value to 0 the closer the sample to the control sample
