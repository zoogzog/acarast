Processing images

There are two ways to process images added to the project.
  • Process only one image, displayed at the Image Viewer
  • Process all images at ones.


The perform first type of processing for the desirable image the Process button () or the corresponding option from the Run menu has to be clicked. This method is the most suitable when it is necessary to tune the parameters of the cell detection algorithm separately for each input file.

The second method allows to set the processing parameters only once and use them for each image from the project. To do that the Process all button () or option from the Run menu has to be clicked.
