Getting started

Processing a file
The easiest way to start using the application and processing image files is to add a file with an Open file button (). Opened file will be displayed in the Image Viewer and could be zoomed in and out with the mouse wheel. To process this file the Process button () or Process option from the menu Run has to be clicked.
During image processing and cell detection it is not allowed to add any new files or samples as well as saving results, however it is possible to view images and already detected samples.

Tool Manager
Tool manager contains the main menu and the toolbar. With the tools from the toolbar and menu user can add files to the project, view the added images, select samples and move them across the image, perform processing and saving the output results.

Button        Name        Description
Open File Creates new project, deletes all previously added files. Opens and adds the specified file to the project.
Add File Adds the specified file to the current project
Open folder Creates new project, deletes all previously added files. Adds all JPG and BMP files from the specified folder to the project.
Zoom In Zooms in currently displayed image.
Zoom Out Zooms out currently displayed image.
Select sample Turns on or off sample selection mode. Samples could be selected by dragging rectangular area on the currently displayed image.
Move sample Turns on or off sample moving mode. If a sample area has been already selected with the Select sample tool, it could be moved by dragging it across the currently displayed image.
Process Performs processing of the currently selected image. If the sample detection mode is auto then if any samples have been previously detected they will be deleted.
Process all Performs processing of the currently selected image. If the sample detection mode is auto then if any samples have been previously detected they will be deleted.
Refresh Resets the view of the currently displayed image, turns off displaying detected cells or samples. Fit image to the Image Viewer area.
View cells Turns on and off displaying detected cells.
Save results Save all output data of the selected image to a new TXT or CSV file.
Save all results Saves all output data to a new TXT or CSV file.
