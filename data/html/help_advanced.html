Advanced Settings

Advanced settings could be accessed by selecting the Settings option in the Help menu. By tuning the settings it is possible to increase the sample and cell detection accuracy for a particular noisy or poorly illuminated sample.

The settings menu is separated into four categories: image processing, sample detection, cell detection and feature extraction. It is suggested that in case if the system could not correctly detect cell samples to change the Sample Detector Settings and if it is necessary to increase the accuracy of cell detection Cell Detector Settings has to be tuned.

Image Processing Settings
  • Gaussian kernel size represents the kernel size in pixels of the gaussian filter which used for correction image illumination. Small values of this parameter could correct illumination more efficiently, however information about some bacteria cells could be lost.
  • Canny edge threshold is the intensity level which used for detecting the edges of the image. Lowering this value could help to detect image cell samples correctly.
  • Dilate kernel size is the size of dilation filter in pixels. Increasing this value could also help to detect cell samples.
  • Local threshold kernel size parameter is used for correctly detecting cells on the image and not treating noise as cells. It is assumed that cells are consist of dark pixels, so if in the area of some pixel, specified by Local threshold kernel size there is not dark pixels which less then Local threshold condition this pixel's intensity is changed to be the Local threshold value.Thus increasing the Local threshold kernel size will make more pixels to be treated as bacteria cells.
  • Local threshold condition represents the intensity level condition for local thresholding (see Local threshold kernel size). Increasing this value will make more pixels to be treated as cells.
  • Local threshold destination value specifies the intensity which will be assigned to a pixel if it does not satisfy the Local threshold condition. It is recommended not to change this parameter. For more information see Local threshold kernel size parameter.
  • Global threshold is the intensity parameter for an ordinary thresholding.

Sample Detector Settings
  • Hough transform rho is a distance parameter of the Hough transform
  • Hough transform theta is an angle parameter of the Hough transform
  • Hough transform threshold is a threshold parameter for lines detected with the Hough transform
  • Hough transform min length is a minimum length of the segment to be detected as line. Increasing this parameter could help to detect sample correctly
  • Hough transform max gap is a maximum gap between segments to be merge. Lowering this value could help to detec sample borders.
  • Offset to narrow sample specifies the amount of pixels on which all borders will be moved to narrow down the sample widths. Narrowing samples is necessary in order to not to detect sample border pixels as cell pixels.
  • Sample border width is the width of sample border in pixels. Setting this parameter correctly will increase accuracy of the sample border estimation
  • Border intensity drop min and Border intensity drop max represent the drop intensity on the border, lowering the minimum and increasing the maximum parameter could help to detect samples.

Cell Detector Settings
  • Min bacteria size specifies the minimum size of cells in pixels. All detected objects with size less then specified will be treated as noise.
  • Max cell distance is the maximum distance in pixels between cells
  • Max angle between cells

Feature Extractor Settings
  • Frequency histogram steps is the amount of bins in the length frequency histogram
  • Confidence interval is the desirable maximum amount of the bacteria cells with certain length in the interval.
